    Full job description

    We are seeking an experienced Unreal C++ Developer to support the development of real-time, high-fidelity simulations, immersive training environments, and interactive prototype systems. The ideal candidate will have a strong foundation in C++ programming and Unreal Engine (UE5 preferred), with a focus on performance, scalability, and modular design. This role involves working closely with designers, engineers, and technical artists to deliver interactive applications with both visual and functional complexity.

    Responsibilities:

    • Develop core systems and gameplay features using C++ in Unreal Engine
    • Create, integrate, and optimize custom plugins, components, and actor classes
    • Collaborate with design teams to implement interactive mechanics, AI behaviors, and system logic
    • Work with Blueprints as needed, while maintaining a focus on C++ as the primary development language
    • Ensure real-time performance and frame-rate stability through profiling and optimization
    • Support the integration of 3D assets, animations, and visual effects into the engine pipeline
    • Assist with packaging, deployment, and cross-platform compatibility (desktop, VR, mobile)
    • Contribute to technical documentation and knowledge sharing across the development team

    Required Skills and Qualifications:

    • Strong proficiency in C++, with at least 2 years of experience developing in Unreal Engine (UE4/UE5)
    • Experience creating and debugging gameplay systems, user interaction logic, and engine extensions
    • Familiarity with Blueprints, with ability to bridge C++ and visual scripting when appropriate
    • Understanding of performance optimization techniques for real-time 3D applications
    • Experience with source control systems such as Git
    • Comfortable working in Agile or iterative development cycles
    • Strong problem-solving skills and ability to work independently or as part of a team

    Preferred Qualifications:

    • Experience developing for VR/AR/MR using Unreal
    • Familiarity with multiplayer/networking in Unreal Engine
    • Knowledge of physics, animation systems, or procedural generation
    • Experience working with Unreal Motion Graphics (UMG) for UI development
    • Understanding of automated build pipelines and deployment tools

    Technical Stack:

    • Core: C++, Unreal Engine 5
    • Scripting: Blueprint (as needed)
    • Tools: Visual Studio, Rider for Unreal, Perforce or Git
    • Platforms: Windows, Linux, VR headsets (Oculus, Vive), and mobile as applicable
